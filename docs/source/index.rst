Clean My Mac
============

A simple command line utility designed to clean your mac

.. code-block:: bash

    $ cleanmymac

.. program-output:: cleanmymac --version

for a detailed view of the command line interface to **cleanmymac** see :ref:`cli`

.. warning::

    This is work in progress and involves deleting files from your system.
    Use at your own risk!
